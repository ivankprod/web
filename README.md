## Quick local start

1.  **Clone this repo**
	```shell
	git clone https://github.com/ivankprod/web-gatsby-netlify-test.git
	```

2.  **Install all dependencies**
	```shell
	cd web-gatsby-netlify-test/
	npm install
	```

3.  **Open the code and start developing**

	The site is now running at http://localhost:8000
