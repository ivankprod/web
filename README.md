## Quick local start

1.  Clone this repo
	```shell
	git clone https://github.com/ivankprod/web.git
	```

2.  Install all dependencies and run the site
	```shell
	cd web
	npm install
	npm start
	```

3.  Open the code and start developing

	The site is now running at http://localhost:8000
